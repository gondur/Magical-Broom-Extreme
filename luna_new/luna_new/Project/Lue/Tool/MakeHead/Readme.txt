LunaAll.hからLuna.hを作成するツール    Programed Sazanta

使い方
>mkLunah.exe <Input Path> <Output FileName>

第１引数に、LunaAll.hのあるパスを指定します。
第２引数に、作成するファイル名を記入します。(パスを含んでもＯＫ)

抜き出し法則については、ソースを見てください。(汚いですけど)
結構いい加減な検出なので、イレギュラーなヘッダには対応できないと思います。
