=====================================================================
	説明
=====================================================================
　XファイルをSelene/Lue用モデルファイルに変換するツールです。
　このツールで変換されたモデルファイル(*.smf)をSelene/Lueでは読み込み、
　表示する事が可能です。

=====================================================================
	使用方法
=====================================================================
 ConvertX2SMF [input file]
    input directroy : source x file name

=====================================================================
	インストール・アンインストール
=====================================================================
　レジストリは一切いじってないので、自由にコピー・削除してもらってOKです。
