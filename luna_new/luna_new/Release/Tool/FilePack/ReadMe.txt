=====================================================================
	説明
=====================================================================
　フォルダを１つのファイルにパッキングするツールです。
　サブディレクトリも再起的に検索してパッキング可能です。

　パックされたファイルをドラッグすることで
　元のデータを取り出すことも可能です。

=====================================================================
	使用方法
=====================================================================
 FilePack [input directory]
    input directroy : source directory name

=====================================================================
	インストール・アンインストール
=====================================================================
　レジストリは一切いじってないので、自由にコピー・削除してもらってOKです。
