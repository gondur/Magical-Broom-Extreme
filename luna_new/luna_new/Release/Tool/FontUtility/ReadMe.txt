=====================================================================
	説明
=====================================================================
　DirectX用ライブラリLunaで使用可能なフォント生成ツールです。
　標準のフォントではつまらないとかいう場合や、他とはちょっと
　差をつけたいときなどにどないでしょか。

　バッチ処理にも対応していますので、一括処理なども簡単に出来るようになっています。
　ツール上での設定をそのままバッチファイルに出力も出来ますので、
　わざわざ引数を調べてバッチファイルを書く必要もありません。

=====================================================================
	使用方法
=====================================================================
　コンバート元のテキストファイルを指定して"作成"ボタンを押せば自動的に
　画像化されたフォントが出力されます。
　細かい設定などは起動していただければ分かるようになっています。

=====================================================================
	インストール・アンインストール
=====================================================================
　レジストリは一切いじってないので、自由にコピー・削除してもらってOKです。

	変更履歴

Version 1.1.0  2006.10.13

Version 1.0.8  2006.04.19

　・使用しているフォントの抽出がテキストを変更した後でも蓄積したままになってしまう不具合の修正。

　・出力データの拡張子が間違っていた不具合の修正。
　・ＴＧＡ出力の際にαが正しく出力されていなかった不具合の修正。

Version 1.0.6  2005.10.06

　・半角フラグの削除。
　　ソレに伴い幅が正しく処理されない不具合の修正。

Version 1.0.5  2005. 9.27

　・アンチエイリアス処理の変更。
　　　アンチエイリアス使用時にフォントが上下してしまう不具合が緩和されます。
　・サイズの比率変更処理を追加。
　　　指定サイズより小さく作成される一部のフォント対策。
　　　また縦と横の比率を変えたい場合などにも。

Version 1.0.3  2005. 9.05

　・半角文字が含まれて居ないフォントを使用する際に
　　半角文字を内部で生成するかどうかのチェックを作成。

　・作成する文字のアスペクト比を指定できるように。
　　　コレに伴いLFDファイルのフォーマットを変更。

Version 1.0.1  2005. 7.26

　・固定ピッチでフォントを出力する機能を追加しました。

　・バッチファイルの読み込み機能を追加しました。

　・コンバートファイルにコントロールコードが入っていた場合は、無視するようにし
　　ました。

　・キャプションにバージョンを出力するようにしました。

　・色の設定ダイアログにおいて、現在設定されているカラーが初期カラーとして反映
　　されない不具合を修正しました。

　・LFDのキャラクターテーブルに格納できない文字コードが入力された場合に、メモリ
　　破壊を起こす可能性がある不具合を修正しました。
