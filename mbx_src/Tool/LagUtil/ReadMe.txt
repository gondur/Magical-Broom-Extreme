=====================================================================
	説明
=====================================================================
　DirectX8用簡易ライブラリLunaで使用可能なBMPパック用ツールです。
　大量のBMPを置いておくのが嫌な場合や、
　BMPファイルを見られたくない場合に有効です。

=====================================================================
	使用方法
=====================================================================
　基本的に元になるＢＭＰ画像の読み込みとアルファチャンネル用のＢＭＰ画像を
　読み込むだけです。
　一応DnDも対応してあります。
　バッチ処理は特定のフォルダのＢＭＰをすべて一括で処理します。
　また｢sample.bmp｣というファイルがあった場合、自動的に｢sample_a.bmp｣
　と"_a"の付いたファイルがアルファチャンネル用のファイルとして読み込まれます。
　なおファイル名から拡張子を除いたものがデータ名になりますが、
　自動的にすべて小文字に変換されます。

　詳しい仕様はヘルプに記載してあります。

=====================================================================
	インストール・アンインストール
=====================================================================
　レジストリは一切いじってないので、自由にコピー・削除してもらってOKです。
